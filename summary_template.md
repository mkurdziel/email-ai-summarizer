# AI Summary Template

Please provide a comprehensive consolidated summary for the following group of emails, targeting a length of approximately 3 pages. Highlight key information from each if relevant, but provide a coherent overview.

## CRITICAL STRICTNESS INSTRUCTIONS
1. Do NOT make up any information or fill in blanks. All content must be explicitly derived from the emails.
2. If context is missing, verify only what is present.

## CONTENT INSTRUCTIONS
1. **Images**: Include any images that seem highly important to the context of the message (embedded as Markdown images).
2. **Links**: Preserve important links from the original emails.
3. **Events**: If there are any upcoming events mentioned, include a consolidated 'Upcoming Events' calendar section at the end.
4. **Sources**: Include a 'Mailing Lists' section listing the sources of these emails and any sign-up/subscription links found.

---

## Output Format

Please structure your response as follows:

### Executive Summary
[Provide a high-level overview of the main topics discussed in these emails.]

### Detailed Summary
[Provide a detailed breakdown of the content, grouped by topic or thread if appropriate.]

### Action Items (if any)
- [List any action items or deadlines found.]

### Upcoming Events
[Consolidated calendar of events, if applicable.]

### Mailing Lists / Sources
[List of sender emails/domains and subscription links.]
